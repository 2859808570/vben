# 项目安装和运行

## 快速开始

项目安装和运行需要四个步骤：

1. **获取项目代码**
2. **安装依赖**
3. **运行开发服务器**
4. **打包生产版本**

## 安装步骤

### 步骤 1：获取项目代码

```bash
git clone <项目仓库地址>
```

### 步骤 2：安装依赖

进入项目目录并安装依赖：

```bash
cd <项目目录名>
npm i -g corepack
pnpm install
```

**注意**：项目使用 `pnpm` 作为包管理器，需要先启用 `corepack`。

### 步骤 3：运行开发服务器

```bash
pnpm dev
```

启动后，访问开发服务器地址（通常是 `http://localhost:5173`）。

### 步骤 4：打包生产版本

```bash
pnpm build
```

打包完成后，生产文件会输出到 `dist` 目录。

## 项目特性

项目基于 Vue3 + Vite + TypeScript 技术栈，主要特性：

- **最新技术栈**：使用 Vue3/vite 等前端前沿技术开发
- **TypeScript**：应用程序级 JavaScript 的语言
- **主题**：提供多套主题色彩，可配置自定义主题
- **国际化**：内置完善的国际化方案
- **权限**：内置完善的动态路由权限生成方案

## 浏览器支持

- **本地开发**：推荐使用 `Chrome 80+` 浏览器
- **生产环境**：支持现代浏览器，不支持 IE
- 支持的浏览器：
  - Edge（最新 2 个版本）
  - Firefox（最新 2 个版本）
  - Chrome（最新 2 个版本）
  - Safari（最新 2 个版本）

## 常见问题

**Q: 安装依赖失败？**  
A: 确保已安装 Node.js 和启用 corepack：`npm i -g corepack`

**Q: pnpm 命令不存在？**  
A: 运行 `npm i -g corepack` 启用 corepack，它会自动管理 pnpm。

**Q: 开发服务器无法启动？**  
A: 检查端口是否被占用，或查看控制台错误信息。

**Q: 打包失败？**  
A: 确保所有依赖已正确安装，检查 TypeScript 类型错误。
