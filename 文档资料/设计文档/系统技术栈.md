# 技术栈规范

## 前端技术栈

- **框架**: Vue 3
- **构建工具**: Vite
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **UI组件库**: Ant Design Vue
- **图标库**: Lucide Vue Next (`lucide-vue-next`)
- **HTTP客户端**: Axios
- **类型系统**: TypeScript
- **工具库**: VueUse (`@vueuse/core`)
- **日期处理**: dayjs
- **CSS框架**: Tailwind CSS
- **国际化**: vue-i18n
- **包管理**: pnpm
- **项目结构**: Monorepo (Turbo)

## 后端技术栈

- **运行时**: Node.js
- **框架**: NestJS
- **数据库**: MariaDB
- **认证**: JWT (JSON Web Token)

## 移动端技术栈

- **UI组件库**: Vant 4
